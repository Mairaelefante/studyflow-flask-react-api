# AI Task Quality ETL

A Python/pandas ETL project that cleans and standardizes messy task-quality style data for analysis and reporting.


Small ETL-style project using Python and pandas to transform messy raw data into a clean dataset.

## Skills Demonstrated
- ETL concepts
- pandas
- missing values
- duplicate removal
- column standardization
- feature creation

## How to Run
```bash
pip install -r requirements.txt
python src/clean_data.py
```
