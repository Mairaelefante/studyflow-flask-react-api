# Customer Retention Data Science

A data science project using Python, EDA, and scikit-learn to explore churn risk and communicate retention insights.


## Goal
Build a small end-to-end data science workflow: collect/clean data, perform EDA, train a baseline predictive model, evaluate results, and explain limitations clearly.

## Skills Demonstrated
- Python data analysis
- pandas, scikit-learn, matplotlib
- Exploratory data analysis
- Data preprocessing and categorical encoding
- Logistic regression baseline model
- Model evaluation metrics
- Clear stakeholder reporting

## How to Run
```bash
pip install -r requirements.txt
python src/eda.py
python src/churn_model.py
```

## Portfolio Notes
This project is intentionally compact and explainable. It is suitable for demonstrating data science reasoning, not claiming production ML experience.
