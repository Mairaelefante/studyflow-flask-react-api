# Evaluation Rubric

Score each AI response from 1 to 5.

## Correctness
Is the response technically accurate?

## Clarity
Is the response easy to understand and well structured?

## Instruction Following
Does the response follow the user's request and constraints?

## Reasoning Quality
Does the response explain the reasoning clearly and correctly?

## Improvement Value
Does the evaluator identify the issue and suggest a better alternative?
