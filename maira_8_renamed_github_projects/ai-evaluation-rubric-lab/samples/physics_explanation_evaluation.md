# Sample Evaluation: Physics Explanation

## User Task
Explain conservation of energy to a beginner.

## Model Response Being Evaluated
"Energy is conserved because the Hamiltonian is invariant under time translation symmetry according to Noether's theorem."

## Evaluation

### Step 1: Audience Fit
The answer is too advanced for a beginner.

### Step 2: Clarity Issue
It uses technical terms without definitions.

### Step 3: Better Answer
Energy does not disappear; it changes form. For example, when a ball falls, gravitational potential energy becomes kinetic energy.

## Score
Correctness: 4/5  
Clarity: 1/5  
Instruction Following: 2/5  
Reasoning Quality: 3/5

## Summary
The answer is technically related but not adapted to the audience.
