# Sales Insights with SQL and Python

A data analyst portfolio project using Python and SQL/PLSQL-style queries to analyze sales performance and business KPIs.


## Goal
Analyze sales order data, extract business insights, and demonstrate Python analysis plus SQL/PLSQL-style reporting.

## Skills Demonstrated
- Python data analysis with pandas
- SQL aggregation, filtering, and business reporting
- PL/SQL-style stored procedure logic
- Data quality checks and KPI reporting
- Matplotlib visualization
- Stakeholder-friendly summary writing

## How to Run
```bash
pip install -r requirements.txt
python src/analyze_sales.py
```

Outputs are saved in `reports/`.

## Portfolio Notes
This project is designed for Data Analyst roles requiring Python, PL/SQL, reporting, data integrity, and clear communication.
