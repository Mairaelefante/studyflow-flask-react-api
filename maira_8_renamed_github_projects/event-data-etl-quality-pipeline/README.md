# Event Data ETL Quality Pipeline

A data engineering project showing extraction, transformation, validation, logging, and schema design for event data.


## Goal
Demonstrate a clean, documented ETL workflow using Python, validation, logging, and SQL schema design.

## Skills Demonstrated
- Python ETL development
- Data cleaning and transformation
- Data quality validation
- Logging and reproducibility
- SQL schema design
- Pipeline documentation
- Data engineering communication

## How to Run
```bash
pip install -r requirements.txt
python src/etl_pipeline.py
```

## Testing
```bash
pip install pytest
pytest
```

## Portfolio Notes
This project supports Data Engineer roles requiring Python, ETL, data quality, documentation, and workflow thinking.
