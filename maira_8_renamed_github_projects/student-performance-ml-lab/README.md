# Student Performance ML Lab

A Python machine learning project focused on predicting and interpreting student performance patterns using pandas and scikit-learn.


End-to-end classification workflow using Python, pandas, NumPy, and scikit-learn.

## Goal
Predict whether a student is likely to pass based on study behavior, attendance, previous score, and support indicators.

## Skills Demonstrated
- Python
- pandas / NumPy
- scikit-learn
- data preprocessing
- feature engineering
- model evaluation
- reproducible ML workflow

## How to Run
```bash
pip install -r requirements.txt
python src/train_model.py
```
