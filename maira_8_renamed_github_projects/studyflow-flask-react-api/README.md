# StudyFlow Flask + React API

A fullstack learning project with a Flask API and React frontend for managing study tasks and technical practice workflows.


Simple fullstack project with Flask REST API and React frontend.

## Skills Demonstrated
- Python / Flask
- REST APIs
- React
- JavaScript
- frontend-backend integration

## Run Backend
```bash
cd backend
pip install -r requirements.txt
python app.py
```

## Run Frontend
```bash
cd frontend
npm install
npm run dev
```
